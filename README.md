# CodingAgents

Simple browser-based calculator built with HTML, CSS, and JavaScript.

## Features
- Add
- Subtract (including `substract` alias in code)
- Multiply
- Divide (with divide-by-zero handling)

## Project Structure
- `index.html`: Complete calculator UI and logic
- `AGENTS.md`: Repository analysis instructions
- `findings/`: Generated findings and onboarding notes

## Run
1. Open `index.html` in a web browser.
2. Enter two numbers.
3. Select an operation.
4. Click **Calculate**.

No install or build step is required.
